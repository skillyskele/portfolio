Context: When is a Moving Average Filter Used?

The goal of this project is to implement a noise-reduction digital signal processing algorithm optimized for embedded systems, focusing on performance, efficiency, and practical implementation in C.

This filter smooths signals by averaging a specified number of points, making it an effective noise-reduction tool. Its simplicity and efficiency make it ideal for resource-constrained embedded systems.

Implementation in C

The implementation in C demonstrates the filter's simplicity and efficiency. Below is the key portion of the code:


#include 

void moving_average_filter(float *input, float *output, int num_samples, int window_size) {
    float accumulator = 0.0;

    // Initialize the accumulator with the first `window_size` points
    for (int i = 0; i < window_size; i++) {
        accumulator += input[i];
    }
    output[window_size / 2] = accumulator / window_size;

    // Sliding window implementation
    for (int i = window_size / 2 + 1; i < num_samples - window_size / 2; i++) {
        accumulator = accumulator - input[i - window_size / 2 - 1] + input[i + window_size / 2];
        output[i] = accumulator / window_size;
    }
}
